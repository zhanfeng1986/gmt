Common Options (Classic Mode)¶

STANDARDIZED COMMAND LINE OPTIONS  
-Binformation Specify map frame and axes parameters (…)
-Jparameters Select map projection (…)
-K Append more PS later (…)
-O This is an overlay plot (…)
-P Select Portrait orientation (…)
-Rwest/east/south/north[/zmin/zmax][+r][+uunit] Specify region of interest (…)
-U[label][+c][+jjust][+odx/dy] Plot time-stamp on plot (…)
-V[verbosity] Run in verbose mode (…)
-X[a|c|f|r]xshift Shift plot origin in x-direction (…)
-Y[a|c|f|r]yshift Shift plot origin in y-direction (…)
-a[col=]name[,…] Associates aspatial data with columns (…)
-bi[ncols][type][w][+l|b] Select binary input (…)
-bo[ncols][type][w][+l|b] Select binary output (…)
-d[i|o]nodata Replace columns with nodata with NaN (…)
-e[~]”pattern” | -e[~]/regexp/[i] Filter data records that match the given pattern (…)
-f[i|o]colinfo Set formatting of ASCII input or output (…)
-g[a]x|y|d|X|Y|D|[col]zgap[+n|p] Segment data by detecting gaps (…)
-h[i|o][n][+c][+d][+msegheader][+rremark][+ttitle] ASCII [I|O] tables have header record[s] (…)
-icols[+l][+sscale][+ooffset][,…][,t[word]] Selection of input columns (…)
-je|f|g Mode of spherical distance calculation (…)
-n[b|c|l|n][+a][+bBC][+c][+tthreshold] Set grid interpolation mode (…)
-ocols[,…][,t[word]] Selection of output columns (…)
-p[x|y|z]azim[/elev[/zlevel]][+wlon0/lat0[/z0]][+vx0/y0] Control 3-D perspective view (…)
-q[i|o][~]rows[+ccol][+a|f|s] Selection of input or output rows (…)
-r[g|p] Sets grid registration (…)
-s[cols][+a|r] Control treatment of NaN records (…)
-ttransparency Set layer transparency (…)
-x[[-]n] Set number of cores in multi-threaded modules (…)
-:[i|o] Expect y/x input rather than x/y (…)
